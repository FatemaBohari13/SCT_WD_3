🎮 Task 3 – Advanced Tic Tac Toe Web Game Application                  

## 📌 Project Overview  
This project was developed as part of my **Web Development Internship**.  
It is an **Advanced Tic Tac Toe Web Application** built using **HTML, CSS, and JavaScript**.
The objective of this task was to implement game logic, DOM manipulation, and AI functionality while creating a responsive and interactive user interface.
The application includes multiple game modes, AI difficulty levels, animations, and a tournament system.


## 🚀 Key Features  
✔ Player vs Player (PvP) Mode  
✔ Player vs CPU Mode  
✔ Three Difficulty Levels (Easy, Medium, Hard)  
✔ Smart AI using Minimax Algorithm  
✔ Scoreboard Tracking  
✔ Tournament Mode (First to 3 Wins) 🏆  
✔ Winning Cell Highlight Animation  
✔ Responsive Design for All Devices  
✔ Smooth UI Transitions  


## 🧠 AI Difficulty Levels  
🔹 **Easy Mode** – Random CPU moves  
🔹 **Medium Mode** – Mix of random and smart moves  
🔹 **Hard Mode** – Unbeatable AI using Minimax Algorithm  


## 🛠️ Technologies Used  
- HTML5  
- CSS3 (Grid, Animations, Responsive Design)  
- JavaScript (Game Logic, DOM Manipulation, Minimax Algorithm)
